## 📖 Overview: The "Director Agent" Pattern

The **AI Bilingual Podcast Generator** is a sophisticated production tool designed to bridge the gap between prompt-based AI and high-quality audio output. It demonstrates a complete **"AI Agentic" workflow** where the LLM acts as a director and producer in a 6-stage creative pipeline.

## ✨ Key Architectural Features

- **🎭 Automated AI Casting:** Orchestration logic that analyzes scripts to dynamically "cast" AI voices based on character personality and tone.
- **🌐 Bilingual-First Synthesis:** Specialized narrative patterns optimized for dual-language content, ensuring educational flow and narrative coherence.
- **📊 Cost & Token Governance:** (See `services/costService.ts`) Built-in transparency for API usage with real-time unit-economic modeling and cost estimation for every generation step.
- **⚡ Resilient State Management:** Custom "Strategy-based" recovery logic to handle LLM truncation or formatting quirks gracefully in asynchronous workflows.
- **✅ Automated Quality Assurance:** Validation logic to detect audio artifacts or TTS truncation using "words-to-time" ratio analysis.

## 🏗 High-Level Architecture

The system follows a multi-stage **Refinement & Synthesis** lifecycle:

1. **Blueprint Generation:** Discovery of themes and cultural context.
2. **Narrative Scripting:** Immersive multi-character dialogue generation.
3. **Casting:** Assigning age/gender/tone-appropriate AI voices.
4. **Optimization:** Refining scripts for audio flow and linguistic accuracy.
5. **Multimodal Synthesis:** Concurrent generation of character segments.
6. **Assembly:** Backend orchestration of music, cues, and final encoding.

## 🛠 Tech Stack (Reference)

- **Frontend:** React 19, TypeScript.
- **Backend:** Python (FastAPI), Pydub, Google GenAI SDK.
- **AI Models:** Gemini 3.1 Pro (Director), Gemini 2.5 Pro TTS (Multimodal Synthesis).
